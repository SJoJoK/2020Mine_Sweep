# 具体实现

1. 软件框架

	软件整体采用MVVM的模式框架，达到多人高效协同的效果。

1. 绘图

	绘图部分采用Qt库的QMainWindow类。

1. 线路规划

	公交的路线规划算法采用优化的Dijkstra算法，对抽象的公交站点地图运用Dijkstra算法，
	并根据总站点数、换乘次数等计算得到不同的公交方案。
